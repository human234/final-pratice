## 遊戲操作說明
滑鼠左右移動控制 Paddle，根據碰撞位置改變反彈角度，使 Ball 擊碎 Block。

## 特色介紹
- **物理真實性**：Block 需要多次擊中才會消失，並在消失時產生爆炸特效，碰撞邏輯貼近真實物理。
- **美術風格**：富有特色的視覺設計，營造出令人沉浸的遊戲體驗。
